github搜索开源项目

1、从名字中搜关键词
in:name example

2、搜索readme文件
in:readme example

3、描述中搜索关键词
in:description example

4、追加限定条件
stars:>3000 # 限定在stars数量大于3000的项目
fork:>50 # fork 数量
pushed:>2019-09-10 # 最后一次push的时间
language:java # 限定语言
